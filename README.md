### 项目结构
**/origin**  
[此项目](https://github.com/mahavivo/english-wordlists)整理的各词表  
并修正几处微小错误  
**/assets**  
从原始词表中提取词头，并做去重处理使得各级间不重复  
**/scripts**  
脚本

_CEFR数据来源于牛津10，一个词条下面各释义的CEFR级别，各级间不做去重_
